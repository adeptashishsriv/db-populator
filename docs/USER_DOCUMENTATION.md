# DB Explorer: Feature Gallery & User Guide

DB Explorer is a lightweight yet powerful database management tool. This guide serves as a visual tour of its core capabilities.

---

## 1. Multi-Tab SQL Workspace
The heart of DB Explorer. Write, execute, and manage multiple queries simultaneously with ease.

![Screenshot: SQL Editor with Syntax Highlighting and Multi-Tabs]

*   **Syntax Highlighting:** Auto-colorized SQL for better readability.
*   **Auto-Completion:** Intelligent suggestions for tables and columns.
*   **Multi-Tab Interface:** Work on multiple databases or queries at once.

---

## 2. Interactive Schema Explorer
Navigate your database structure with a simple tree-based browser.

![Screenshot: Schema Explorer Tree showing Tables, Views, and Procedures]

*   **Tree Navigation:** Expand to see Tables, Views, Procedures, and Functions.
*   **Metadata at a Glance:** Hover or expand columns to see data types and constraints.
*   **Context Actions:** Right-click to quickly generate SELECT statements or export schema.

---

## 3. Advanced Query Results & Data Viewer
View your data in a high-performance, interactive grid.

![Screenshot: Results Grid with Pagination and Column Sorting]

*   **Fast Loading:** Handles large datasets with efficient pagination.
*   **Sorting & Filtering:** Click column headers to sort data instantly.
*   **Cell Selection:** Copy individual values or entire rows.

---

## 4. Database Health Dashboard
Monitor your connection's pulse in real-time.

![Screenshot: Health Dashboard showing active connections and performance metrics]

*   **Live Metrics:** Track active sessions and query performance.
*   **Connection Status:** Visual indicators (Green/Amber/Red) for connection stability.

---

## 5. Visual Schema Diagrams
Visualize table relationships without leaving the app.

![Screenshot: Schema Diagram showing Table Relationships]

*   **Entity-Relationship View:** See how your tables connect via Foreign Keys.
*   **Interactive Layout:** Move tables around to better understand complex schemas.

---

## 6. Explain Plan Analyzer
Optimize your queries by understanding how the database executes them.

![Screenshot: Explain Plan Visualization]

*   **Visual Execution Plan:** See bottlenecks and scan types (Index vs. Full Table).
*   **Performance Tuning:** Get insights to speed up slow-running queries.

---

## 7. Versatile Export Tools
Get your data out in the format you need.

![Screenshot: Export Dialog showing CSV, SQL INSERT, and DDL options]

*   **Formats:** Export to CSV, JSON, or SQL (INSERT/UPDATE).
*   **DDL Export:** Generate CREATE TABLE scripts for existing structures.

---

## 8. Memory Management & UI Customization
Stay in control of your environment.

![Screenshot: Status Bar with Heap Monitor and Theme Toggle]

*   **Memory Monitor:** Real-time JVM heap usage tracking in the status bar.
*   **Garbage Collection:** Manual 🗑 button to free up memory during heavy use.
*   **Modern Themes:** Toggle between Light, Dark, and High-Contrast modes.

---

## 9. Quick Setup: Connection Wizard
Connecting to a new database takes seconds.

![Screenshot: New Connection Dialog with Database Type Dropdown]

*   **Native Support:** PostgreSQL, MySQL, SQL Server, Oracle, SQLite, and DynamoDB.
*   **Test Connection:** Verify credentials before saving.
